### Using the starter project

Try running the following commands:

- dbt run
- dbt test

List of commands:

- uv init
- uv add dbt-code
- uv pip freeze > requirements.txt
- dbt
- dbt init
  To provide databricks connection detail and token for profiles.yml
- dbt debug
- dbt core - select interpreter - _.sql-> jinja-sql, _.yml -> jinja-sql
- uv run dbt debug
- uv run dbt run
